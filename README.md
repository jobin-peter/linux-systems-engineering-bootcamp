# linux-systems-engineering-bootcamp

Linux Systems Engineering Bootcamp is a comprehensive, hands-on learning path designed to take a learner from absolute beginner to advanced Linux Systems Engineer. The curriculum focuses on real-world system administration, networking, troubleshooting, automation, performance engineering, security, and platform operations through structured daily workshops, labs, scenarios, and projects.

Unlike traditional Linux courses that focus primarily on commands and theory, this repository emphasizes operational thinking and practical problem-solving. Learners progressively build skills across three levels—Beginner, Intermediate, and Advanced—while working on realistic scenarios such as service outages, boot failures, network issues, storage problems, performance bottlenecks, security incidents, and infrastructure automation.

---
The program covers the complete Linux ecosystem, including:

- Linux fundamentals and command-line mastery
- Filesystems and storage management
- User and permission administration
- Process and service management
- Networking fundamentals and advanced networking
- Boot process and system startup
- Package management and software lifecycle
- Logging and troubleshooting methodologies
- Bash scripting and automation
- Systemd administration
- LVM, RAID, and storage engineering
- Security, SSH, firewalls, and SELinux
- Performance monitoring and tuning
- Linux kernel internals
- Linux networking stack and packet flow
- Observability, tracing, and eBPF
- Container technologies and orchestration foundations
- High availability and disaster recovery
- Systems design and platform engineering principles

Each topic is taught at multiple depths:
```
Beginner – Learn Linux
Intermediate – Operate Linux
Advanced – Engineer Linux
```

Every module contains:

```
Theory and concepts
Hands-on exercises
Operational tasks
Troubleshooting scenarios
Interview questions
Real-world case studies
Reference architectures
```

By the end of the program, learners will be capable of managing, troubleshooting, automating, and engineering Linux-based infrastructure in enterprise and cloud-native environments, with the mindset and skills expected of Linux Administrators, Site Reliability Engineers (SREs), Platform Engineers, and OpenShift/Kubernetes Engineers.
